🎓 Udemy Clone (Basic)

This project is a simple Udemy-inspired website built using HTML and CSS. It is designed as a beginner-friendly front-end project to practice layout design, styling, and responsive web development.

🚀 Features
📌 Clean and modern UI similar to Udemy
📚 Course listing section
🧭 Navigation bar with logo and menu
🎨 Styled using pure CSS
📱 Basic responsive design (for different screen sizes)
🛠️ Technologies Used
HTML5
CSS3
📂 Project Structure
udemy-clone/
│
├── index.html
├── style.css
├── images/
└── README.md
💡 Purpose of the Project

This project was created to:

Practice HTML structure and semantic tags
Improve CSS styling skills
Understand layout techniques like Flexbox/Grid
Build a real-world UI clone as a beginner
